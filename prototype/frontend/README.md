# Installation
1. Install with package.json
2. Start with npm start
3. Server startet auf port 3000

# Konfiguration

Es wird eine Verbindung zur ICDD Plattform benötigt. Dafür muss eine Instanz der RUB ICDD Plattform gehostet werden. Diese ist über folgendes GitHub Repository zu beziehen: https://github.com/RUB-Informatik-im-Bauwesen/icdd-platform 

 RUB ICDD Plattform benötigt folgende Abhängigkeit: https://github.com/RUB-Informatik-im-Bauwesen/IIB.ICDD 

 Eine Schnittstellendokumentation ist hier zu finden: https://github.com/RUB-Informatik-im-Bauwesen/open-icdd

 Neben dem Frontend werden weitere services benötigt, die ebenfalls in diesem Repository vorhanden sind und nach Anleitung gestartet werden müssen:

 * https://github.com/RUB-Informatik-im-Bauwesen/greeninfratwins/tree/main/prototype/calculation-service
